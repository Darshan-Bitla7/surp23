ET02

  • Description:In this project, the objective is to develope an mobile application to teach No-Code AI tools. The mobile app should act as an repository to find a suitable No-Code AI tool for the query given by the user and shows how to execute the task using that No Code AI tool.
  • Number of students: 2
  • Year of study: Students entering 2nd year, Students entering 3rd year, Students entering 4th/5th year
  • CPI eligibility criteria: none
  • Prerequisites: You will be responsible for designing, developing, testing and deploying mobile applications for various platforms. Proficiency in one or more programming languages, such as Java, Kotlin, Swift, C#, JavaScript. Familiarity with mobile application development tools and environments, such as Xcode, Android Studio, Visual Studio or Flutter. Experience in using version control systems, such as Git.
  • Duration: Minimum 2 months (summer vacation). After that as long as the candidate is willing to work.
  • Learning outcome: 1. Understanding how a mobile app works. 2. Understanding how No-Code AI works.
  • Weekly time commitment: 4 hours/ week
  • General expectations: You are going to be the primary developer of the app. It is going to be used by n number of students. This is a live app targeted for showcasing at TechConnect - Techfest 2023. This is not a dummy project. So unless you are seriously interested, please do not apply.
  • Assignment: https://www.sciencedirect.com/science/article/pii/S0007681323000502
  • Instructions for assignment: Read the paper. We will discuss this during interview.

ET03

  • Description:In this project, the objective is to develop a No-Code AI Research Tech Stack model. You will be using a combination of various no-code AI tools to create a tech stack that can perform human like research based on a query from the user.
  • Number of students: 2
  • Year of study: Students entering 3rd year, Students entering 4th/5th year
  • CPI eligibility criteria: NONE
  • Prerequisites: We are looking for a software engineer who can combine multiple APIs to create a no code tech stack. You will be responsible for designing, developing and testing solutions that integrate various services and platforms without writing complex code. - Experience with API integration, RESTful web services and JSON. Proficiency in one or more programming languages such as Python, JavaScript, Ruby or PHP. Familiarity with no code tools and platforms. Knowledge of web development frameworks and technologies such as React, Angular, Node.js or Django.
  • Duration: 6 months to 1 year. Students can have option to exit after 6 months.
  • Learning outcome: 1. You will know how to seamlessly connect multiple APIs towards executing a given task. 2. Understand how different no-code AI tools work.
  • Weekly time commitment: 4 hour/week
  • General expectations: This is a tentative project which may be done in collaboration with Amazon. Our target is to send a paper in IJCAI 2024 or AIED 2024. This is not a dummy project. So apply only if you are genuinely interested.
  • Assignment: "1. https://www.sciencedirect.com/science/article/pii/S0007681323000502 2. https://www.mdpi.com/2079-9292/10/10/1192 3. https://dl.acm.org/doi/fullHtml/10.1145/3568026"
